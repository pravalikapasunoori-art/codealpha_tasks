# CodeAlpha IoT Internship

## Task 2: Sensor-Based Simulation

### Project
Motion Sensor Based Automatic LED Control System using Arduino

### Files
- motion_sensor_led_control1.ino
- circuit.png
- simulation_output.png

### Description
This project uses a PIR motion sensor to detect movement. When motion is detected, the LED turns ON. When no motion is detected, the LED remains OFF.

### Components
- Arduino Uno
- PIR Motion Sensor
- LED
- 220Ω Resistor
- Breadboard
- Jumper Wires
